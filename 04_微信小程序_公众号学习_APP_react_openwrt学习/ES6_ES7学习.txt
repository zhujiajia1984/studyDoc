ES6的let和const命令：https://www.w3cschool.cn/ecmascript/ueqp1q5g.html
1：let：类似于var，但是所声明的变量，只在let命令所在的代码块内有效。（块作用域）
   之前只有全局作用域和函数作用域。
2：const声明一个只读的常量。一旦声明，常量的值就不能改变。JSON对象内容可变，不变的指向的内存地址
3：ES5只有两种声明变量的方法：var命令和function命令。ES6除了添加let和const命令，后面章节还会提到，另外两种声明变量的方法：import命令和class命令。所以，ES6一共有6种声明变量的方法。

ES6的变量解析赋值：
1：var [a, b, c] = [1, 2, 3]; 相当于var a = 1;var b = 2;var c = 3;
2：数组中对应赋值。
3：对象的解构赋值：对象的解构与数组有一个重要的不同。数组的元素是按次序排列的，变量的取值由它的位置决定；而对象的属性没有次序，变量必须与属性同名，才能取到正确的值。
var { bar, foo } = { foo: "aaa", bar: "bbb" };
foo // "aaa"
bar // "bbb"
